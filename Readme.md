# 24-Point Game (猜 24 點)

這是一個基於 JavaScript和C++ 開發的益智小遊戲。玩家需要利用系統隨機給出的 4 個數字，透過加、減、乘、除運算，最終湊出 24。

## 🎮 遊戲規則
1. 系統每局會隨機產生 4 個數字（1-13，對應撲克牌 A-K）。
2. 玩家必須使用這 4 個數字各一次。
3. 使用 `+`, `-`, `*`, `/` , `(` , `)`符號。
4. 使算式結果等於 24。

## 🛠️ 技術棧
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Answer:** C++ 
- **Deployment:** GitHub Pages

## 🚀 如何執行
只需瀏覽：https://glen0414.github.io/24-Game/

或是下載到本地開啟：
1. `git@github.com:glen0414/24-Game.git`
2. 用瀏覽器打開 `index.html`
